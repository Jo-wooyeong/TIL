## v-for
반복 대상에 v-for 디렉티브를 붙이고 필수로 :key 속성바인딩을 기입해야합니다. (v-bind를 줄여 :)

```ts
<template>
    <ul v-if="item.length > 0">
        <li v-for="item in items" :key="item.id">
            {{ item }}
        </li>
    </ul>
</template>

<script setup>
import { ref } from 'vue';

const items = ref([
  { id: 101, text: 'Vue 공부하기' },
  { id: 102, text: '리액트와 비교하기' },
  { id: 103, text: '마스터 되기' }
]);
</script>
```
