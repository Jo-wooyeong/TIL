## v-if
v-if true이면 DOM에 해당 디렉티브가 있는 태그를 생성합니다.<br />
이런 점에서 v-show 와 차이가 있습니다.

### v-if와 v-for를 한 태그안에 사용하지 못한다!
틀린 문법
```ts
// 틀린 문법
<template>
    <div v-if="item.length > 0" v-for="item in items" :key="item.id">
        {{ item }}
    </div>
</template>
```

정답 문법 - 계층 나누기
```ts
// 수정 문법
<template>
    <ul v-if="item.length > 0">
        <li v-for="item in items" :key="item.id">
            {{ item }}
        </li>
    </ul>
</template>
```

## v-show
v-if와 용도가 비슷하지만 값에 따라 display="none/block"으로 구분됩니다.
장점은 초기 비용이 많이 들지만 반응이 즉각적입니다.

```ts
<template>
    <button v-show="inVisable">버튼</button>
</template>
```
