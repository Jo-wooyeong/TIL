## v-on
축약해서 `@`을 붙여 사용합니다.

### @click
클릭이벤트

```vue
<button @click="handleClick">클릭</button>

<button @click="deleteItem(item.id)">삭제</button>

<button @click="count++">증가</button>
```

### @mouseenter, @mouseleave
마우스 호버시 사용하는 디렉티브

```vue
<div 
  @mouseenter="isHovered = true" 
  @mouseleave="isHovered = false"
  :class="{ active: isHovered }"
>
  여기에 마우스를 올려보세요!
</div>
```

### 이벤트 수식어
`.prevent`, `.stop` 을 사용해서 html 레벨에서 직접 처리할 수 있습니다.
- `.prevent`: e.preventDefault() 실행 (주로 form 전송 막기)
- `.stop`: e.stopPropagation() 실행 (버블링 방지)
- `.once`: 이벤트를 딱 한 번만 실행

```vue
<form @submit.prevent="saveData"> ... </form>

<div @click.stop="handleChildClick"> ... </div>
```
