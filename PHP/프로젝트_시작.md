## 프로젝트 시작
```
Lalavel로 만든 프로젝트는 php언어를 사용한다.
프로젝트 초기 세팅 시 해야할 몇 가지 규칙을 알고 실행시켜야 웹사이트를 똑바로 확인할 수 있다.
```

### .env파일 생성
데이터베이스 정보, 환경변수 등이 작성되어있는 `.sql`파일을 생성한다.

### 마이그레이트
설계도를 바탕으로 DB를 생성한다.
```
php artisan migrate
```

### DB 삽입
초기 데이터(관리자 계정, 더미 데이터 등) 삽입.
```
php artisan db:seed
```
- `php artisan migrate:fresh --seed` 를 사용하면 DB를 초기화하고 데이터까지 삽입할 수 있음
