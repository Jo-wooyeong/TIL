## reduce()
```
배열의 각 요소를 하나씩 돌면서, 최종적으로 하나의 결과값을 만들어내는 아주 강력한 함수입니다.
단어 뜻 그대로 배열을 '줄여가며(reduce)' 하나의 값으로 합치는 과정
```

인자에 콜백함수가 들어가는 콜백함수의 인자로 `누적값`과 `현재값`이 차례로 필수로 들어갑니다.<br />
return으로 계산을 하면 다음 `누적값`으로 전달 <br />
- (세번쨰는 현재 인덱스, 네번째는 원본배열을 담을 수 있고 선택입니다.)
```js
const numbers = [1, 2, 3, 4];

const sum = numbers.reduce((acc, cur) => {
  console.log(`누적값: ${acc}, 현재값: ${cur}`);
  return acc + cur;
}, 0);

console.log(sum); // 결과: 10
```
